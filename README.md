# APK Ticket System

سیستم مدیریت تیکت - تسک FullStack APK (امن پردازان کویر)

## ساختار پروژه

```
ticket-system/
├── backend/     # Django + DRF
├── frontend/    # React + shadcn + Tailwind
├── docker-compose.yml
└── README.md
```

## مجوزها (Permissions)

### کاربر عادی (User)
- تیکت‌های خود را می‌بیند
- می‌تواند تیکت جدید بسازد
- می‌تواند به تیکت‌های خود پاسخ دهد
- می‌تواند تیکت خود را حذف کند (فقط مالک)

### ادمین (Admin / is_staff)
- **همه تیکت‌ها را می‌بیند**
- **می‌تواند به تیکت‌ها پاسخ دهد**
- **می‌تواند وضعیت تیکت را تغییر دهد** (open, in_progress, closed)
- **نمی‌تواند تیکت‌ها را حذف کند** — فقط مالک تیکت اجازه حذف دارد
